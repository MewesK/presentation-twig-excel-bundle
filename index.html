Pure PhpExcel

$objPHPExcel = new PHPExcel();
$objPHPExcel->setActiveSheetIndex(0);
$objPHPExcel->getActiveSheet()->setTitle('Worksheet');
$objPHPExcel->getActiveSheet()->getStyle('A1')->getFont()->setSize(18);
$objPHPExcel->getActiveSheet()->setCellValue('A1', 'Values');
for ($i = 0; $i < count($data); $i++) {
  $objPHPExcel->getActiveSheet()->SetCellValue('B' . $i, $data[i]);
}
$objWriter = new PHPExcel_Writer_Excel2007($objPHPExcel);
$objWriter->save(str_replace('.php', '.xlsx', __FILE__));

So how do we integrate this mess into our Symfony project?

We could put it in our controller.

But that way we lose our controller/view separation.

So how do we integrate this mess into our view layer?

We could use PHP as our templating language.

But that way our view layer isn't as clean and secure.

So why don't we extend Twig to support Excel creation?

I know - crazy!

Twigified PhpExcel

{% xlsdocument %}
  {% xlssheet 'Worksheet' %}
    {% xlsrow %}
      {% xlscell { style: { font: { size: '18' } } } %}Values{% endxlscell %}
    {% endxlsrow %}
    {% for value in data %}
      {% xlsrow %}
        {% xlscell %}{{ value }}{% endxlscell %}
      {% endxlsrow %}
    {% endfor %}
  {% endxlssheet %}
{% endxlsdocument %}

Benefits

  • Pure Twig (10 custom tags, over 100 properties)
  • Built-in caching
  • Supports creation of XLS, XLSX, CSV and PDF files
  • Smooth Symfony integration
  • Configurable with @Route and @Template
  • Controller actions can be reused

Controller example

class HelloController
{
  /**
   * @Route("/hello.{_format}", defaults={"_format"="xls"}, requirements={"_format"="csv|xls|xlsx"})
   * @Template("AcmeHelloBundle:Hello:index.xls.twig")
   */
  public function indexAction($name)
  {
    return array('data' => array('La', 'Le', 'Lu'));
  }
}

Footer / Header example

  • No more messing around with &L, &C and &R
{% xlsheader %}
  {% xlsleft %}Left part{% endxlsleft %}
  {% xlscenter %}Center part{% endxlscenter %}
  {% xlsright %}Right part{% endxlsright %}
{% endxlsheader %}

Drawing example

  • Can be used in headers / footers
{% xlsdrawing '/test.png' {
  description: 'Test',
  offsetX: 10,
  offsetY: 10
} %}

Bonus: How it works

->write('$rowIndex = ')->subcompile($this->getNode('index'))->raw(';'.PHP_EOL)
->write('$phpExcel->startRow($rowIndex);'.PHP_EOL)
->subcompile($this->getNode('body'))
->write('$phpExcel->endRow();'.PHP_EOL);

Bonus: How it works

->write('$cellIndex = ')->subcompile($this->getNode('index'))->raw(';'.PHP_EOL)
->write("ob_start();\n")
->subcompile($this->getNode('body'))
->write('$cellValue = trim(ob_get_clean());'.PHP_EOL)
->write('$cellProperties = ')->subcompile($this->getNode('properties'))->raw(';'.PHP_EOL)
->write('$phpExcel->startCell($cellIndex, $cellValue, $cellProperties);'.PHP_EOL)
->write('$phpExcel->endCell();'.PHP_EOL);
